Author Manager is a must-have for administrators of a multiple-author Wordpress site. Easily view post statistics by date.

== Description ==

This plugin makes it very simple for site administrators to keep track of how many posts contributors, authors, editors and administrators have published. Admins can choose a week from the drop-down menu, or input a specific date range

**Additional Features**

* Easily show/hide posts 
* Post titles are linked to posts for quick viewing
* Viewable only by administrators
* Simple interface


== Installation ==

1. Download 'author-manager.zip' and upload the contents to the /wp-content/plugins/ directory **OR** upload 'author-manager.zip' through the 'Plugins' upload section in the WordPress administrator.

2. Activate the plugin.

3. Access the Author Manager interface under Users->Author Manager.
